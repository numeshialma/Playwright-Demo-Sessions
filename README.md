# Playwright-Demo-Sessions

> Hands-on learning project for Playwright automation using TypeScript.

---

##  About
This project is designed to gain practical experience with **Playwright** and **TypeScript**.  
It includes basic to intermediate test scenarios, structured using the **Page Object Model (POM)**, with a focus on writing maintainable and scalable automation scripts.
